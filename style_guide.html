Layout

Grid

The layout of the page is build using the bootstrap 3 vertical 12-column grid. I removed the 30px gutters that is in the default bootstrap framework. The framework offers a good foundation for responsive design fit for this exercise. The breaking points are set to a "max-width: 768px" for mobiles, "max-width: 992px" for tablets and "992px and up for desktops and larger desktops at 1200px". I've set a certain css content to break at 960px which I think is suitable for this design.

Filter

The .filter-bar is a full width div containing the three type of elements: The button to launch the #filter modal, the .filter-wrap divs with the .eventicon and .icon-text and lastly the 960px+ visible compact view radio toggle buttons.

Eventicons

Eventicons are square divs with a centered letter within that serve as a "type of event" icon. There are 7 type of eventicons and each has a unique colour and letter. The desktop view offers the .icon-text along with the eventicon in the filter bar to serve as a legend.

T Travel & Registration
B Breaks & Meals
P PegaWorld main events
M Mobile
U User Centered Design
C Customer Centricity
O Other
O Hidden eventicon styling

Event columns

There are 5 event columns .agenda-col one for each event day. All of them have a 2-grid column in desktop view except for #wednesday-col which has a double sized column. All of them will be shown as a full width 12-grid column below the 992px breakpoint.

All columns consist mainly of an .agenda-label with the date specified and an .agenda-item div that wraps the the event content such as the .agenda-time, .agenda-title and .agenda-place.

Compact view

The compact view toggle is a desktop only feature to help presenters fit the entire week agenda on a screen/projector. This feature is available up to the 992px breakpoint. What this feature does is overwrite the padding of .agenda-content (padding: 10px 5px 10px 35px !important;) to (padding: 5px 5px 5px 35px !important;) using the agenda-content-compact class. It also adds .compact-inline class to have the .agenda-place shown inline of the .agenda-title. Lastly it downsize the font within agenda-content to 12px.

Header

The header .navbar is the top div that carries three elements: The #logo image which is the only external linked file, the Title and the navigation links.

The logo

The #logo is external hosted. This is the only external hosted file that is linked to the document. The element will be hidden in case of an error upon retrieval. The logo has a set height of 40px.

The page title

The #header-title div houses the "PEGAWORLD 2015" Header h1 title and Header p subtitle.

Navigation

The navigation consist of a ul with hyperlinks. These links do nothing but enable on.click jQuery events which filters the agenda-col to suit the prefered view. Below the breakpoint of 992px this will turn into a .navbar-toggle dropdown menu

Buttons

The filter categories button uses the pega blue to set itself apart from the other button sets. The purpose for this is to have the user notice this as a primairy button. The styling uses the default bootstrap btn styling with a custom colour styling using the .btn-pega class.

This button-group shows a set of two radio buttons styled as buttons. It is used to give the user the impression of a toggle.

Modal

The modal renders using custom css overwriting the standard boostrap 3 modal styling. The close button on the right top corner to fit the nav-toggle styling. The close modal-dimission link ("close") has a much larger surface thant he original button for improved accuracy.
